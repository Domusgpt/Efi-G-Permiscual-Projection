# apps/web

User studio and admin console.

## First screens

- Landing page.
- Auth shell.
- Private studio dashboard.
- Avatar creation form.
- Prompt composer.
- Generation job status.
- Private gallery.
- Creator waitlist.
- Admin review queue.

## UI principle

The user sees a simple creative studio. The system quietly enforces verification, rights scope, policy, credit, and audit requirements before generation.
